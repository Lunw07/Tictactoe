# Tic-Tac-Toe 

## A **Tic-Tac-Toe game** built with **Python** and **Tkinter**.

This project is a Tic-Tac-Toe game with single-player and multi-player. You can play against another player on 3×3, 4×4, or 5×5 boards, challenge an AI opponent with Easy, Medium, or Hard difficulty using the Minimax algorithm, and track your scores with a leaderboard system.

---

## Features

- Play against another player on 3x3, 4x4, or 5x5 boards
- Single-player mode with an AI opponent using Minimax algorithm with Easy, Medium and Hard difficulty
- Leaderboard system to track your highest scores when playing against AI
- Intuitive graphical interface with clickable buttons
- Automatic detection of wins and draws

---

## How to Use

1. Make sure you have **Python 3** installed.
2. Tkinter is required (usually comes pre-installed with Python).
3. Download or clone this repository.
4. Run the game by running main.py
